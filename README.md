## Project Description: 
This project is a personal portfolio that showcases my web development skills. It is designed to highlight my experience, skills, and technical abilities. This portfolio includes various sections such as **About Me**, **Skills**, **Experience**, and **Contact**, all styled using **HTML5** and **CSS**. 

## **Technologies Used**:
- **HTML5**: Used to structure the content and layout of the website.
- **CSS**: Applied for styling, including Flexbox for layout, CSS animations, and media queries for responsive design.
- **GitHub**: For version control and hosting the project files.

## File Organization:
- 'Masood_Aman_Portfolio.html': Main HTML file located at the root level.
- 'css/styles.css': External CSS file placed within the 'css/' folder for better file organization.
- 'README.md': Project description file.

## Features Implemented:
- **Responsive Design**: The website is fully responsive and adapts to different screen sizes using media queries.
- **CSS Animations**: Smooth fade-in effect for main content, hover effects for buttons, and transitions on interactive elements.
- **Centering Content**: Ensured that all text and sections are centered appropriately.
- **Accessible HTML**: Proper use of semantic HTML elements.
- **Custom Properties**: Defined CSS variables for easy customization of colors, fonts, and spacing.

## Implementation Notes:
- The project was designed using a mobile-first approach with media queries to ensure good performance and layout on mobile, tablet, and desktop devices.
- The table displaying my **Technical Skills** is responsive and ensures a good user experience across various devices.
- The profile image is circular and centered as required by the project specifications.
